# PongKdor.com - Sprint Plan: Week 1

**Sprint Goal**: Set up the foundation — Branding, Design System, and Initial Project Structure.

**Duration**: 7 days (can be adjusted based on your availability)

**Focus**: Preparation + Strong start (not rushing into coding everything)

---

## Day 1: Branding & Visual Foundation

**Tasks**:
- Finalize logo (primary + secondary version)
- Define color palette (Black, Red, Gold + supporting colors)
- Choose typography (heading + body fonts)
- Create basic brand guidelines (in Figma or Notion)

**Deliverables**:
- Logo files (SVG + PNG)
- Color palette documented
- Font pairings chosen

---

## Day 2: Figma Design System Setup

**Tasks**:
- Create new Figma file for PongKdor
- Set up Design System structure:
  - Colors
  - Typography
  - Spacing system
  - Button components (multiple variants + states)
  - Card components
  - Input fields
- Create reusable components with variants and auto-layout

**Deliverables**:
- Well-organized Figma file with component library started

---

## Day 3: High-Level Page Wireframes

**Tasks**:
- Create wireframes for key pages:
  - Homepage
  - Blog Listing
  - Article Detail
  - Shop Collection
  - Product Detail
- Focus on layout, hierarchy, and user flow (not high-fidelity yet)

**Deliverables**:
- Wireframe screens for main pages

---

## Day 4: High-Fidelity Homepage Design

**Tasks**:
- Design the full Homepage in high fidelity
- Include:
  - Hero section with animated feel
  - Featured memes / content section
  - Shop teaser
  - Footer
- Apply the design system

**Deliverables**:
- High-fidelity Homepage design

---

## Day 5: Project Setup (Technical)

**Tasks**:
- Initialize Next.js 15 project with TypeScript + Tailwind
- Set up folder structure based on Clean Architecture
- Install core dependencies:
  - Framer Motion
  - shadcn/ui
  - React Hook Form + Zod
  - Sanity client
  - Shopify Hydrogen (or Storefront API)
- Connect domain to Vercel (if ready)
- Set up GitHub repository

**Deliverables**:
- Working Next.js project on GitHub + Vercel

---

## Day 6: Content Planning + First Articles

**Tasks**:
- Finalize first 5–8 article topics
- Write outlines for 3 articles
- Create content structure in Sanity (schemas)

**Deliverables**:
- Article topics list
- Sanity schema ready

---

## Day 7: Review + Planning for Week 2

**Tasks**:
- Review everything done in Week 1
- Fix any issues
- Plan Week 2 tasks (start building pages + connecting data)

**Deliverables**:
- Clean foundation ready for development

---

## Week 1 Success Criteria

- [ ] Logo + Brand identity finalized
- [ ] Figma design system started
- [ ] Homepage high-fidelity designed
- [ ] Next.js project initialized and deployed
- [ ] First content structure ready in Sanity

**End of Week 1 Sprint Plan**
